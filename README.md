This system is about Topic 2: Network Security and Protocol Analysis

It allows users to run a simulator of the following:
  1. Malicious word detector (Involves DFA)
  2. TCP handshake validation (Involves PDA)

To run the system in terminal 

  " g++ main.cpp -o main "       (Used to compile)
  
  " ./main "                     (To run)
  
Optional: To run this system you need "Graphviz" to execute the .Dot files enabling you to see visual representation of the system
